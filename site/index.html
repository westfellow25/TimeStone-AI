How it works

From your financials to a CFO-ready PDF in one command.

1

Digital twin

Capture revenue, segments, competitors, prior transformations and strategic priorities. Five enriched Kazakhstan / global twins ship in the repo.

2

Scenario generation

1,000 transformation hypotheses are generated, each calibrated against the most similar real cases in the library — not generic templates.

3

Monte Carlo + NPV

5-year NPV at configurable WACC. Adoption ramp-up, market downturn, competitive response, execution failure modelled per iteration.

4

Empirical priors

Per scenario: top-5 similar real cases inform the probability distribution. Their actual revenue uplift and failure rate ground the math.

5

Top recommendations

Ranked by P(NPV>0). Each with mean NPV, ROI, payback, and citations to the cases that informed the prior.

6

Executive PDF

5-page A4 report with cover, summary, recommendations, methodology, sources. Ready to hand to a CFO.

The moat compounds

Every engagement makes the next forecast more accurate.

Generic AI tools cannot replicate this. The product becomes more valuable with every recorded outcome:

  • Curated case library with promised-vs-actual outcomes
  • Per-engagement OutcomeRecord tracking — append-only
  • Bayesian calibration loop adjusts priors after each measurement
  • Local CIS / Kazakhstan case library that no public LLM can extract
  • Failure-mode aware retrieval — learn from same mistake

Try the demo

Five seed companies are bundled. One command produces a full assessment with executive PDF.

# Install
git clone https://github.com/westfellow25/timestone-ai
cd timestone-ai
pip install -e .

# List bundled twins
python -m timestone list-companies

# Run a full assessment
python -m timestone assess "Kazakhstan Temir Zholy (KTZ)"

# Open the resulting PDF
open results/runs/<run_id>/report.pdf

# Or launch the interactive dashboard
streamlit run src/timestone/interfaces/web/dashboard.py

**Tool: Loom (free)** · loom.com → record screen + camera

**Total length:** 90 seconds, divided into 6 beats.

**Before recording:**
- Open https://westfellow25.github.io/timestone-ai/ in Chrome, full screen
- Press Ctrl+Shift+R to clear cache
- Switch language to EN if international audience, RU if Kazakhstani CEOs
- Close all other tabs / notifications

## Tips for the recording

- **Voice over is more important than perfect screen movement.** Conversational
  pacing, slight pauses between sentences.
- **Don't zoom or move the cursor erratically.** Slow, deliberate movements.
- **When the simulation runs, let it complete on screen** — don't try to talk
  over fast-moving animations.
- **If you flub a beat, re-record only that beat** — Loom lets you trim.
- **Cap viewers' attention to the screen** — the Loom face bubble can sit
  bottom-right, small.
- **Export as MP4** for sharing in pitch decks, LinkedIn, email.

## After recording

- Upload to Loom (gives you a shareable link)
- Add the Loom link to:
  1. README of GitHub repo (top of file)
  2. Slide 1 of pitch deck (replace the byline)
  3. LinkedIn post when announcing
  4. Cold-outreach email to investors

If the Loom link looks too long, shorten via https://shrtco.de or use
Loom's built-in custom alias.
